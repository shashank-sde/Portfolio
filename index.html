This web project is a blog project and is designed to be static, meaning the page loads everything during the initial load itself. It has 3 blogs in it which gets loaded from the JSON file based Database integrated, within the code. When a blog is clicked it gets fully opened up and gets displayed on the same page without reloading. The website also has an API section in which we make API calls to websites after every particular duration of time and the content refreshes automatically.

This is a website designed using the 3 column layout from general layout and the 4 column layout coming to the food section. It is designed keeping in mind the need for a restaurant. The cards display each of the food item being offered by the restaurant. The items can be purchased using the buttons provided and they show the accurate count of how many respective items have been purchased. This required Javascript funcionality to achieve. However, this website can be further improved a lot by including menu, services etc.
